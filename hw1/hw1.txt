Change directory: cd pathToDirectory
Create a directory: mkdir pathToNewDirectory
Rename a directory: mv oldDirectoryName newDirectoryName
List contents of directory: ls -lart [directoryPath]
Move a file to a new location: mv oldFilePath newFilePath
Copy a file to a new location: cp oldFilePath newFilePath
Search for a string in a file: grep 'string' fileName
Access the shell of a remote machine: ssh user@remotehost
